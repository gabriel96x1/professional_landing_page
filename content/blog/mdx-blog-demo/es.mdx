Este articulo confirma que el blog ya lee entradas reales desde archivos MDX locales. La metadata vive junto al contenido y la pagina puede renderizar Markdown enriquecido desde el repositorio.

## Que valida este post

El flujo nuevo prueba tres superficies importantes:

- La seccion de blog en el home muestra entradas reales.
- El indice del blog lista metadata tomada del MDX.
- La ruta individual renderiza el cuerpo completo del articulo.

## Como se escribe una entrada

Cada idioma tiene su propio archivo MDX y su propio slug. Eso mantiene las URLs naturales para cada audiencia y deja un `id` estable para conectar traducciones.

```ts
export const metadata = {
  id: "mdx-blog-demo",
  slug: "probando-blog-con-mdx",
};
```
